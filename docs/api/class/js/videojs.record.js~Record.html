    public saveAs(name: object) source

    Show save as dialog in browser so the user can store the recorded media locally.

    Params:

    NameTypeAttributeDescription
    name object

    Object with one or more names for the particular blob(s) you want to save. File extensions are added automatically. For example: {'video': 'name-of-video-file'}. Supported keys are 'audio', 'video' and 'gif'.

    Example:

    // save video file as 'foo.webm'
player.record().saveAs({'video': 'foo'});
